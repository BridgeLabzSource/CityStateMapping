Indian State Finder

[![Build Status](https://travis-ci.org/joemccann/dillinger.svg?branch=master)](https://travis-ci.org/joemccann/dillinger)

It helps you to get the state name on the basis of the city name or latitude and longitude provided.

## Features

- Correct misspelt city name to best possible correctly spelt city name using     nltk
- uses Nominatim library to retrieve the state
- works for both either city name or coordinates as input

## Installation

Install required dependencies and then can use the package

pip install find_indian_state
